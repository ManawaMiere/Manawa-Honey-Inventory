Export & import

Export a workbook with live stock (grouped + by batch) and the full transaction history — or import stock from a spreadsheet.

Layout chooses how stock is laid out: grouped rolls each product into one line; Each batch separate gives every batch its own row (so Puatane 3kg comes out as its 3 batches). The batches sheet always lists each batch by name. Import reads a sheet with columns like Product name, Batch, Jar size, Waitawa, Mataatua, Reorder point, MGO, Blend, Comments — re-importing a file exported here works directly.

Data storage

Local keeps data on this device. Cloud sync shares stock across every phone and computer in real time using your Supabase project.

Use the publishable key (or legacy anon key) — never the secret / service_role key.

Run supabase_setup.sql in your Supabase SQL editor once before connecting — it creates the tables and loads your products.

WooCommerce

Push finished stock to your website. Each product needs its WooCommerce product ID set (edit a product to add it). Requires cloud sync and the woo-sync Edge Function deployed — see WOOCOMMERCE_SETUP.md.
